# 단기 일용직 구인/구직 플랫폼 '인력거' 
## 0.개요
’인력거’는 기존 오프라인 인력사무소의 대면 구인/구직 프로세스에 불편함을 느껴 고안한 사업 아이템으로,
인력사무소 측에서 ‘인력거’ 애플리케이션을 통하여 구인 광고를 업로드하고
일용직 근로자는 현장에 지원할 수 있는 비대면 구인/구직 애플리케이션
